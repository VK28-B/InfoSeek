Privacy Policy

Last updated: January 2026

Application Name: InfoSeek

Developer: Vishwajeet Kumar Vaybhav

1. Introduction:
InfoSeek is a desktop application designed to provide fast, distraction-free access to Wikipedia content. This Privacy Policy explains how the application handles data and user information.
By using InfoSeek, you agree to the practices described in this policy.

2. Information We Collect:
InfoSeek does not collect, transmit, or sell any personal data.
The application only stores the following information locally on your device:
-Search queries entered by the user
-Search mode used (Summary or Detailed)
-Date and time of searches
User preferences such as:
-Theme (light or dark)
-Font size
-Reading mode
-Speech rate
This data is stored in a local SQLite database on your system and never leaves your device.

3. Information We Do Not Collect:
InfoSeek does not collect or access:
Name, email address, or personal identity
Location data
Contacts or files outside the app
Keystrokes outside the application
Browsing activity
Camera, microphone, or background system data

4. Internet Usage:
InfoSeek requires an active internet connection only to fetch content from Wikipedia.
Searches are sent directly to Wikipedia servers using the Wikipedia API
No search data is sent to the developer
InfoSeek does not track or log network activity externally
Wikipedia’s own privacy policy applies when accessing their services.

5. Data Storage and Retention:
All stored data remains on your local device
Search history is automatically deleted after 30 days
Users can manually delete individual searches or clear the entire history at any time

6. Data Security:
InfoSeek does not use cloud storage or external servers.
Because all data is stored locally, security depends on the user’s operating system and device protections.

7. Third-Party Services:
InfoSeek uses the following third-party libraries and services:
Wikipedia API for content retrieval
Text-to-Speech engine provided by the operating system
These services operate independently and are governed by their own privacy policies.

8. Children’s Privacy:
InfoSeek does not knowingly collect personal information from children. Since no personal data is collected, the app is safe for general audiences.
